## 4. Servidores Web

En esta unidad aprenderemos a instalar, configurar y administrar servidores web modernos, con especial enfoque en Nginx. Cubriremos desde la instalación básica hasta la configuración de sitios virtuales, seguridad HTTPS y transferencia de archivos mediante SFTP.

Esta unidad cubre los **Resultados de Aprendizaje 2 y 3** del módulo:

- **RA2**: Implanta aplicaciones web en servidores web, evaluando y aplicando criterios de configuración para su funcionamiento seguro
- **RA3**: Administra servidores web, analizando y aplicando criterios de configuración y seguridad
